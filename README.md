# 📹 Screen Recorder

Screen এর উপর সবসময় একটা floating icon থাকে।
Click করলে record শুরু, আবার click করলে বন্ধ — app এ ঢুকতে হয় না!

## ✅ Features
- Floating icon — সবসময় screen এ থাকে
- Drag করে যেকোনো জায়গায় সরানো যায়
- একটা click এ record শুরু/বন্ধ
- Gallery → Movies → ScreenRecorder এ MP4 save
- No watermark • No internet • No ads

## 🚀 GitHub Actions দিয়ে APK বানাও

1. এই repo তে code push করো
2. GitHub এ **Actions** tab এ যাও
3. **"Build APK"** → **"Run workflow"** চাপো
4. Build শেষে **Artifacts** থেকে APK download করো

## 📱 ব্যবহার

1. APK install করো
2. App খোলো → দুইটা permission দাও
3. App বন্ধ হবে, screen এ 🟢 icon আসবে
4. Icon click → 🔴 Record শুরু
5. আবার click → ✅ Gallery তে Save
